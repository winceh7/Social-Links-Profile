### What I learned

During the development of this project I focused on simplifying the code as much as posssible, specially CSS by observing areas where general styling can be applied to elements and areas where specific styling is necessary, in order for it to be more readable. As well, trying to use a combination of HTML semantic elements fror accessibility along with non-semantic elements used as wrappers for styling and scripting to have a more structured, 'modular' code. 

```html
<section class="profile-info">
        <img src="./assets/images/JPSuarez_AvatarCut-resized-to-176x176.jpeg" alt="Profile picture" class="profile-pic">
        <div>
          <h1 id="name">Juan Pablo Suarez</h1>
          <h3 id="location">Guadalajara, Mexico</h3>
        </div>
        <p id="description">Game & Level designer and aspiring UX Engineer.</p>
      </section>
```
One part of my HTML code where I used the combination of semantic and non-semantic HTML elements.
